About Lusorio's
Pixel Sorter

LPS is a tool to create pixel sorted images. It works defining bright and dark areas in the picture, that then get sorted. It is written in JavaScript, HTML and CSS with the Electron Framework.

Features

Current features are loading a PNG or JPG image. Threshold, shadows or highlights, and mask oise can then be tweaked with a live preview. Finally the resulting image can be saved as either PNG or JPG.

Planned

A planned feature is using custom masks.
Suggestions for other features are welcome.
