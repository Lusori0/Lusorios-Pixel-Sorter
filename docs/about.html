About Lusorio's
Pixel Sorter

LPS is a tool to create pixel sorted images. It works by defining areas in the picture that should get sorted and then sorting the pixels in those areas after brightness. Lusorio's Pixel Sorter™ is written in Python 3.8 and makes heavy use of the Pillow and TKinter libraries.

Features:

Current features only include being able to set a threshold and choosing wether the highlights or the shadows should be targetted. However I am planning on implementing new features like being able to import custom masks and creating animation, so stay tuned for those features.
